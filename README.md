📱 Me Chat – Real-Time Flutter Chat App

Me Chat is a real-time messaging app built with Flutter, powered by Firebase for authentication and real-time chat, and Supabase for media storage. It enables seamless communication with a beautiful UI, secure login, and media sharing support.
🚀 Features
🔐 Firebase Authentication (Email/Password)
💬 Real-time messaging using Firestore
🖼️ Media sharing via Supabase Storage
🎯 Clean, responsive UI for Android & iOS
🌐 Cross-platform (mobile + web-ready architecture)
🛠️ Tech Stack
Layer	Tool/Service
UI	Flutter (Dart)
Auth	Firebase Auth
Realtime DB	Firebase Firestore
Media	Supabase Storage
